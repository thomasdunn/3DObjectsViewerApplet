3D Objects Viewer Applet

Please use a browser that can view Java 1.1 applets, such as Netscape Navigator 4.0+ or Internet Explorer 4.0+

Instructions

  • Choose a model from the models selection box.
  • If "Fill Object?" is checked, the model will be filled and shaded. Otherwise the model will be a "wireframe".
  • If "Rotate Object?" is checked, the model will automatically rotate.
  • You can rotate the model with your mouse by dragging the model.
  • Some models are displayed as wireframes by default. The faces of these models do not all show up correctly.
    They are still interesting to view filled though. Click the Fill Object checkbox to fill them.
  • Some models, ones with hundreds of faces and vertices, can take a while to load.
  • For slow loading models, status can be seen in the browsers status area, at the bottom of the browser window.
